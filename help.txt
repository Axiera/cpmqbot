join [название очереди] - присоединиться к очереди
leave [название очереди] - покинуть очередь
info [название очереди] - получить список участников очереди
q list - получить список очередей
skip [название очереди] - поменяться местами с человеком, идущим в очереди за вами
url [ссылка] - установить ссылку (например на доску), которая будет автоматически отправляться консультанту; вызов этой команды со словом "del" вместо ссылки удаляет ссылку; рекомендуется отправлять боту в личные сообщения

Команды для администраторов:
admin [имя пользователя] - назначить администратора
unadmin [имя пользователя] - удалить администратора
create [название очереди] - создать очередь
delete [название очереди] - удалить очередь
next [название очереди] - пригласить пользователя из очереди
time [время в секундах] - установить время ожидания пользовтеля после команды next

После выполнения команды next при наличии подходящих голосовых комнат:
1) Администратор автоматически переносится в подходяющую комнату, при условии, что он уже подключен к одной любой голосовой конмате сервера
2) Участник очереди переносится в подходящую комнату, при условии, что он уже подключен к голосовой комнате "ожидание"
